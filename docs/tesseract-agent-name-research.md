# Tesseract Agent — Name Availability Research

> Research date: 2026-03-10

## Summary

**"Tesseract Agent"** is **not currently used** by any app on the Mac App Store or iOS App Store. However, the word "Tesseract" alone is heavily contested — multiple apps and trademarks exist. The addition of "Agent" differentiates the name sufficiently for App Store listing, but trademark risk exists in the AI/robotics category due to an active filing by Tesseract Ventures, LLC.

| **Mobelux Labs, LLC** | Reg 4404838 | 009 | Downloadable photo viewing/sharing software | Possibly dead | LOW — narrow scope, likely abandoned |

### Key Risk: Tesseract Ventures, LLC

This is the most relevant trademark holder. Their filing covers:
- "Humanoid robots with artificial intelligence"
- "Software for controlling and interfacing with robots"

An AI agent app could be argued as adjacent to "software for controlling AI." However:
- Their mark is **"TESSERACT"** alone, not "Tesseract Agent"
- Their goods are physical robots and robot-control software, not consumer AI chat apps
- Class 009 is broad, but the specific goods description is robot-focused
- **No "Tesseract Agent" mark exists in the USPTO**

### No Trademark Found For:
- "Tesseract Agent" (exact phrase) — **clear**
- Tesseract OCR (open-source, Apache-licensed, no trademark filing found)
- Any AI assistant or chatbot named "Tesseract"

## 5. Apple App Store Naming Rules

Per Apple's [App Store Review Guidelines](https://developer.apple.com/app-store/review/guidelines/):

- **30 character limit** for app names ("Tesseract Agent" = 16 chars — well within limit)
- **No promotional phrases** ("free", "best", "#1")
- **No other app/brand names** in your title
- **No keyword stuffing** — name must be the actual app name
- **Must be unique enough** to avoid confusion with existing apps
- Subtitle field available (30 chars) for additional description

"Tesseract Agent" passes all these rules cleanly.

| Apple rejection for name | **LOW** | Name is clean, descriptive, 16 chars, within all guidelines |

| Domain squatting | **NONE** | All `tesseractagent.*` domains are currently available |

## 7. Recommendation

**"Tesseract Agent" is a viable name for App Store release.** Key reasons:

1. No existing app or trademark uses this exact name
2. The word "Tesseract" is a common geometric/mathematical term (4D hypercube), making broad trademark claims difficult
3. The "Agent" qualifier clearly differentiates from all existing "Tesseract" apps (games, journals, business tools)
4. The name accurately describes the product (an AI agent)
5. At 16 characters, it fits comfortably within App Store limits
6. We already own `thetesseract.app` for web presence

**Before finalizing:**
- [ ] Run a manual USPTO TESS search at [tmsearch.uspto.gov](https://tmsearch.uspto.gov) for "TESSERACT AGENT" to confirm no recent filings
- [ ] Register `tesseractagent.com` and/or `tesseractagent.app` (both available as of 2026-03-10)
- [ ] Consider a brief consultation with a trademark attorney given the Tesseract Ventures filing and Cisco overlap
- [ ] Reserve the App Store name in App Store Connect early (names are first-come-first-served)
- [ ] Consider filing a Class 9 trademark application for "Tesseract Agent" (downloadable computer software) to protect the name long-term

## 8. Rename Scope

Changing from "tesse-ract" to "Tesseract Agent" requires updates in:

| Category | Count | Key Files |
|----------|-------|-----------|
| Display name (`CFBundleDisplayName`) | 2 | `project.pbxproj` (Debug + Release) |
| Bundle identifier | 5 | `project.pbxproj` (app, tests, UI tests) |
| Logging subsystem | 9+ | `Logging.swift`, vendor loggers |
| User-facing strings | 6 | `MenuBarManager.swift`, `OnboardingView.swift` |
| Data directory paths | 4 | `PathSandbox.swift`, `AgentConversationStore.swift`, `TranscriptionHistory.swift`, `ContextLoader.swift` |
| Scripts | 5 | `dev.sh`, `bench.sh`, `tts-headless.sh`, etc. |
| Documentation | 4 | `CLAUDE.md`, `DISTRIBUTION.md`, `TESSE_DEVELOPMENT_PLAN.md`, `APP_STORE_METADATA.md` |
| Xcode project/scheme | 60+ | `project.pbxproj`, `tesseract.xcscheme` |

**Note on bundle ID**: Changing from `com.tesseract.app` should be done carefully. A new bundle ID means:
- Existing sandbox containers won't carry over
- Any provisioning profiles need to be regenerated
- App Store Connect record needs a new entry

Suggested new bundle ID: `com.tesseract.agent` or keep `com.tesseract.app` and only change the display name.
